# EL ToDo REACT
## Tutorial instalación
### Paso 1. Descargar y ponerlo en una carpeta bien localizada.
### Paso 2. Abrir la cmd y desplazarnos a la carpeta en cuestión.
### Paso 3. Hacer un "npm run dev" para poder iniciar el proyecto, este nos proporcionara un enlace (https://localhost:5173).
### Paso 4. Tienes la aplicación para su libre uso.
